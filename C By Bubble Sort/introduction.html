  1. 動畫呈現

    2. 透過動畫讓使用者了解泡沫排序的交換方式

  2. 給使用者練習（狀況一：使用者輸入正確）

    3. 使用者輸入完成後，按下提交，當輸入正確時，
    會跳出警示框跟使用者提示

  3. 給使用者練習（狀況二：使用者輸入錯誤）

    4. 使用者輸入完成後，按下提交，當輸入錯誤時，
    會跳出警示框跟使用者提示，並且會標記錯誤的位置

  4. 全部完成

    5. 當使用者全部都完成後，會給使用者一個完整的泡沫排序程式碼，
    該程式碼正是使用者前面全部所學的內容
